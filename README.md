# Earthquake Damage Prediction System

This application predicts building damage levels after an earthquake using a Machine Learning model.

## Model Used
- XGBoost Classifier
- Scikit-learn Pipeline

## Features
- End-to-end ML workflow
- Data preprocessing pipeline
- Streamlit web application
- Real-time prediction

## Damage Levels
1 - Low Damage
2 - Medium Damage
3 - Severe Damage
